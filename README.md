# Mechanical Engineering Tools

Dieses Repository enthält nützliche Python-Skripte für Berechnungen im Maschinenbau-Studium.

## Biegespannungs-Rechner
Ein Tool zur schnellen Dimensionierung von Balkenquerschnitten.

### Features
* **Berechnung:** Bestimmung der Biegespannung basierend auf Moment, Breite und Höhe.
* **Optimierung:** Automatische Berechnung der minimalen Balkenhöhe (`while`-Loop) zur Einhaltung der Sicherheitsnormen.
* **Material-Datenbank:** Unterstützung für S235, S355 und Aluminium.

### Verwendung
1. Skript in JupyterLab ausführen.
2. Parameter (Breite, Höhe, Moment) eingeben.
3. Ergebnis für die Dimensionierung erhalten.
